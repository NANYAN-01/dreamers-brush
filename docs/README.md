# 童梦画笔 (Dreamer's Brush)

> AI 绘本插画生成器

完整文档请查看 [docs/README.md](docs/README.md)

## 快速开始

```bash
npm install
npm run dev        # 启动前端开发服务器
npm run server     # 启动后端服务器
```

## 项目结构

- `client/` - React 前端应用
- `server/` - Express 后端 API
- `database/` - 数据库迁移和脚本
- `docs/` - 项目文档
- `scripts/` - 测试工具脚本
- `archive/` - 归档的废弃代码
